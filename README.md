# Hospital Cleaning Optimization
A tool for cleaners and hospital staff to see room cleanliness based on heat sensors.

# Built with
* Swift
* Xcode version 11.3
* Mongo DB
* Gitlab CI/CD
* iOS deployment target 13.2
